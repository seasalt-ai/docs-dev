SeaVoice Overview¶

Table of Contents

  • SeaVoice SDK (Python)

  • SeaVoice RESTful API

The purpose of this tutorial is to demonstrate how to use SeaVoice SDK and services to do speech recognition and speech synthesis, and to customize your own Speech-To-Text (STT) models and Text-To-Speech (TTS) rules. In this tutorial we will walk-through: - How to use the SeaVoice SDK (Python) to do STT and TTS - How to use the SeaVoice RESTful APIs to do STT and TTS - How to use the STT customization feature to customize your STT model - How to use the TTS customization feature to customize your TTS rules
